# Rules Refactor Working Set

This directory contains planning documents for restructuring the DSAT rule
system without modifying the original source rule files in the repository root.

Contents:

- `rules_refactor_plan.md` - concrete migration plan from the current two-file
  state to a three-document rules architecture
- `rules_core_generation_outline.md` - outline for the shared core generation
  rules document
- `rules_grammar_module_outline.md` - outline for the grammar-domain module
- `rules_reading_module_outline.md` - outline for the reading-domain module

Source documents reviewed:

- `rules_agent_dsat_grammar_ingestion_generation_v3.md`
- `rules_agent_dsat_reading_v1.md`

Constraint:

- The original documents remain unchanged. These files are planning artifacts
  only.
