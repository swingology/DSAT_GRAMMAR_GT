### Question 1

### Question 4
**Answer: C) 10**

**Explanation:** Since triangle EDC has sides 6 and 3 (ratio 2:1), CE = 2 × AC. Let AC = x, then CE = 2x. Since AE = AC + CE = x + 2x = 3x = 15, we get x = 5. Therefore CE = 2x = **10**.

### Question 6
**Answer: A) x = yz − w**

**Explanation:** Starting from y = (x + w)/z:
1. Multiply both sides by z: yz = x + w
2. Subtract w: **x = yz − w**

### Question 8
**Answer: C) a > 0 and at least one of b or c is negative**

**Explanation:** Since the parabola opens upward, **a > 0** (eliminating A and B). The vertex coordinates are both negative (in quadrant III). In factored form f(x) = a(x − b)(x − c), the x-intercepts are b and c — at least one must be negative for the vertex (their midpoint) to be negative. If both b and c were positive (as in D), the vertex x-value would be positive. So **C** is correct.

### Question 9
**Answer: B) Exactly one solution**

**Explanation:** Convert both equations to slope-intercept form:
- 2x + 6y = 20 → y = −(1/3)x + 10/3 (slope = −1/3)
- 4x + 2y = 20 → y = −2x + 10 (slope = −2)

Different slopes and different y-intercepts → the lines intersect at exactly **one point**.

### Question 10
**Answer: −6**

**Explanation:** Isolate the absolute value:
|2x + 6| + 4 = 8 → |2x + 6| = 4

Set up two equations:
- 2x + 6 = 4 → 2x = −2 → x = **−1**
- 2x + 6 = −4 → 2x = −10 → x = **−5**

Sum: −1 + (−5) = **−6**

### Question 11
**Answer: B) 3x − y ≤ 3 (equivalent to y ≥ 3x − 3... verify as y ≤ 3x + 3)**

**Explanation:** The line has slope 3 and y-intercept 3, so the boundary is y = 3x + 3. The region is shaded **below** (less than or equal to). The correct inequality is **y ≤ 3x + 3**. Among the answer choices, after rearranging option B (−y ≥ −3x − 3 → y ≤ 3x + 3), **B** is correct.

### Question 12
**Answer: D) y = −16(x − 0.25)² + 6**

**Explanation:** The vertex form of a parabola is y = a(x − h)² + k where (h, k) is the vertex. The vertex is (0.25, 6), so the equation is y = −16(x − 0.25)² + 6. Verify: at x = 0.25, y = −16(0)² + 6 = **6** ✓. Answer is **D**.

### Question 13
**Answer: D) 64 times**

**Explanation:** The area formula for a trapezoid is A = ½(b₁ + b₂)h. If all dimensions are multiplied by 8, then:
- (b₁ + b₂) scales by 8
- h scales by 8
- Area scales by 8 × 8 = **64**

### Question 15

**Explanation:** (Note: The SAT had a printing error; the intended expression was (√(2q) + √(2r))^(2/3).)

Rewrite as the cube root of the square:
= ∛[(√(2q) + √(2r))²]
= ∛[2q + 2√(2q·2r) + 2r]
= ∛[2q + 2·2√(qr) + 2r]
= **∛(2q + 4√(qr) + 2r)**
