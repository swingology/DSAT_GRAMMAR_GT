### Question 1 — Combining Like Terms

**Correct Answer: C**
**Topic:** Algebra — Polynomial Simplification
**Skill:** Identifying and combining like terms

In the expression $6x + 5x + 4y$, the terms $6x$ and $5x$ share the variable $x$ and can be combined:
$$6x + 5x + 4y = 11x + 4y$$

> [!tip] Strategy: Only Combine Like Terms
> Like terms must have **identical variables and exponents**. $6x$ and $5x$ are like terms; $4y$ is not. Combine only like terms.

> [!danger] Trap: Combining Unlike Terms
> Don't add $11x + 4y = 15xy$ — they have different variables. Only the numeric coefficients of identical variable terms can be added.

### Question 7 — Exponential Function — Initial Value (Gridded)

**Correct Answer: 3,600**
**Topic:** Advanced Math — Exponential Functions
**Skill:** Finding the initial value of an exponential model

The initial bacteria count is the value of $y$ when $x = 0$:
$$y = 3600 \cdot a^0 = 3600 \cdot 1 = 3{,}600$$

Since any positive base raised to the 0 power equals 1, the initial count is the coefficient **3,600**.

> [!tip] Strategy: Plug in x = 0
> In any exponential model $y = a \cdot b^x$, the initial value is $a$ because $b^0 = 1$.

> [!danger] Trap: Confusing the Base and Initial Value
> In $y = 3600 \cdot a^x$, the initial value is **3600** (coefficient), not $a$ (the base/growth factor).

### Question 8 — Writing Inequalities from Context

**Correct Answer: A ($1.87c + 2.40t \leq 15$)**
**Topic:** Algebra — Modeling with Inequalities
**Skill:** Translating word problems into algebraic inequalities

- Each container: \$1.87 → total cost for $c$ containers: $1.87c$
- Each roll of tape: \$2.40 → total cost for $t$ rolls: $2.40t$
- Leo has at most \$15:

$$1.87c + 2.40t \leq 15$$

> [!tip] Strategy: Parse "At Most" → ≤
> "Leo has \$15" means the total cost must be **less than or equal to** \$15. "At most" always becomes $\leq$.

> [!danger] Trap: Flipping the Inequality Direction
> "At most \$15" → ≤ 15 (cost is bounded above). Choosing ≥ 15 represents "at least \$15," which is the opposite.

### Question 9 — Identifying Function Type from Graph

**Correct Answer: B (Increasing Exponential)**
**Topic:** Advanced Math — Function Behavior
**Skill:** Classifying functions by their graphical behavior

The graph shows $f(x)$ increasing as $x$ increases (not decreasing), and the rate of increase is growing (curved upward), not constant. This describes an **increasing exponential** function.

> [!tip] Strategy: Check Two Properties
> 1. Is it increasing or decreasing as x grows?
> 2. Is the rate of change constant (linear) or growing/shrinking (exponential/quadratic)?
> 
> Curved + increasing rate → **exponential**. Straight line → **linear**.

> [!danger] Trap: Confusing Linear and Exponential Growth
> Both can be increasing. The key difference: exponential curves **accelerate** (the slope itself increases), while linear graphs have a **constant slope** (straight line).

### Question 11 — Exponential Decay Model (Tennis Tournament)

**Correct Answer: C ($p = 240 \cdot (1/2)^r$)**
**Topic:** Advanced Math — Exponential Decay
**Skill:** Writing a decreasing exponential model from context

![[assets/q11_tennis_decay.png]]

With 240 players, each round eliminates half the remaining players. The number eliminated after round $r$:
$$p = 240 \cdot \left(\frac{1}{2}\right)^r$$

> [!tip] Strategy: Identify the Decay Factor
> "Decreases by half each round" → multiply by $\frac{1}{2}$ per round. Starting value is 240. Model: $p = 240 \cdot (1/2)^r$.

> [!danger] Trap: Using a Growth Model Instead of Decay
> The number of players is **decreasing** each round — the base must be less than 1. Using $240 \cdot 2^r$ would model doubling, not halving.

### Question 12 — Parallel Lines / Slope

**Correct Answer: A ($y = 6x + 5$)**
**Topic:** Algebra — Linear Equations / Parallel Lines
**Skill:** Using slope of parallel lines

Line $k$: $y = 6x + 4$ → slope = 6

Line $j$ is **parallel** to line $k$ → same slope = 6. Line $j$ passes through $(0, 5)$ → y-intercept = 5:
$$y = 6x + 5$$

> [!tip] Strategy: Parallel Lines Have Equal Slopes
> Copy the slope directly from the parallel line. Then use the given point to find the new y-intercept.

> [!danger] Trap: Using the Wrong y-Intercept
> Line $k$ has y-intercept 4, but line $j$ passes through $(0, 5)$ — a different y-intercept. The parallel constraint only copies the **slope**, not the intercept.

### Question 14 — Circle Radius from Equation (Gridded)

**Correct Answer: 13**
**Topic:** Geometry — Circles
**Skill:** Reading the radius from a circle equation

Standard form: $(x - h)^2 + (y - k)^2 = r^2$

For $(x+2)^2 + (y+5)^2 = 169$:
$$r^2 = 169 \implies r = \sqrt{169} = 13$$

> [!tip] Strategy: Take the Square Root of the Right Side
> The right-hand side of the standard circle equation is $r^2$, not $r$. Always take the square root to find the radius.

> [!danger] Trap: Reporting r² Instead of r
> $r^2 = 169$, so $r = 13$. Reporting 169 is a common mistake — that's the square of the radius.

### Question 16 — Polynomial Zeros and Factors

**Correct Answer: C ($x - 1$)**
**Topic:** Advanced Math — Polynomial Functions
**Skill:** Factor theorem: zeros of polynomials correspond to linear factors

The graph of $y = f(x)$ passes through $(-5, 0)$, $(1, 0)$, and $(4, 0)$.

By the **Factor Theorem**, if $(a, 0)$ is a zero, then $(x - a)$ is a factor:
- Zero at $x = 1$ → factor is $(x - 1)$ ✓

> [!tip] Strategy: Factor Theorem — Zero at x = a Means Factor (x − a)
> For each x-intercept $a$, the corresponding factor is $(x - a)$. Be careful with signs: zero at $x = -5$ gives factor $(x + 5)$, not $(x - 5)$.

> [!danger] Trap: Sign Error on Factors
> Zero at $x = -5$ → factor is $(x - (-5)) = (x + 5)$, not $(x - 5)$. Always negate the zero value to get the factor.

### Question 17 — Linear Function from Table

**Correct Answer: D (b = 36)**
**Topic:** Algebra — Linear Functions
**Skill:** Solving for a constant using two data points

Given $g(x) = mx + b$ with table values:
- $g(1) = 32$: → $m + b = 32$ → $b = 32 - m$
- $g(2) = 28$: → $2m + b = 28$

Substituting $b = 32 - m$ into the second equation:
$$2m + (32 - m) = 28 \implies m = -4$$
$$b = 32 - (-4) = 36$$

> [!tip] Strategy: Use Two Points to Set Up a System
> With $g(x) = mx + b$, plug in two $(x, g(x))$ pairs to get two equations. Then solve the system for $m$ and $b$.

> [!danger] Trap: Stopping After Finding m
> The question asks for $b$, not $m$. After finding $m = -4$, substitute back to get $b = 36$.

### Question 18 — Quadratic Function from Table

**Correct Answer: B ($f(x) = (x-24)(x-30) - 8$)**
**Topic:** Advanced Math — Quadratic Functions
**Skill:** Using the Factor Theorem and given values to determine a quadratic function

From the table: $f(24) = -8$, $f(30) = -8$, $f(32) = 8$.

Define $g(x) = f(x) + 8$. Then $g(24) = 0$ and $g(30) = 0$, so:
$$g(x) = k(x - 24)(x - 30)$$

Using $g(32) = f(32) + 8 = 16$:
$$16 = k(32-24)(32-30) = k(8)(2) = 16k \implies k = 1$$

Therefore: $g(x) = (x-24)(x-30)$, and:
$$f(x) = g(x) - 8 = (x-24)(x-30) - 8$$

> [!tip] Strategy: Shift the Function to Create Zeros
> When multiple table values share the same output (both $f(24)$ and $f(30) = -8$), add a constant to create zeros, factor, then shift back.

> [!danger] Trap: Using f(x) Directly Without the Shift
> Don't try to factor $f(x)$ directly — it doesn't have obvious zeros. Instead, shift by adding 8 to create zeros at $x = 24$ and $x = 30$.

### Question 20 — Radical / Exponent Expression (Gridded)

**Correct Answer: 41/81 (also accepted: 0.5061, 0.5062)**
**Topic:** Advanced Math — Exponent Rules
**Skill:** Converting radical expressions to rational exponents

Key rule: $\sqrt[n]{a^m} = a^{m/n}$

Given equation leads to: $p^{3/2} = p^{7/9 \cdot (3n-1)}$

Setting exponents equal and solving:
$$\frac{3}{2} = \frac{7}{9}(3n - 1)$$
$$\frac{21}{2} = 7(3n - 1) \implies \frac{3}{2} = 3n - 1$$
$$n = \frac{41}{81}$$

> [!tip] Strategy: Convert Radicals to Rational Exponents
> $\sqrt[n]{a^m} = a^{m/n}$. Once both sides have the same base, set the exponents equal and solve.

> [!danger] Trap: Arithmetic Errors in Fractional Exponents
> This problem has several fractions. Write each step carefully and double-check by substituting back.

> [!info] Gridded Response Note
> Enter **41/81**, **.5061**, or **.5062** — all are accepted. The fraction form is most precise.

### Question 21 — Percent Chain Calculation (Gridded)

**Correct Answer: 302.4 (also accepted: 1512/5)**
**Topic:** Problem-Solving & Data Analysis — Percentages
**Skill:** Applying sequential percent changes

Step 1 — Find $b$ (320% greater than 160):
$$b = 160 + 3.2 \times 160 = 160 \times 4.2 = 672$$

Step 2 — Find $a$ (55% less than $b$):
$$a = b - 0.55b = 0.45b = 0.45 \times 672 = 302.4$$

> [!tip] Strategy: "320% greater" ≠ "320% of"
> "320% greater than 160" means $160 + 320\%$ of $160 = 160(1 + 3.2) = 160 \times 4.2$.
> 
> Then "55% less than $b$" means $b \times (1 - 0.55) = 0.45b$.

> [!danger] Trap: "320% greater" vs. "320% of"
> "320% greater than 160" ≠ "320% of 160" (which would be 512). The "greater" means you **add** 320% of 160 to 160 itself.

> [!info] Gridded Response Note
> Enter **302.4** or **1512/5**. Both are valid.

### Question 22 — Vertex Form of Quadratic

**Correct Answer: C ($f(x) = -6(x-5)^2 + 24$)**
**Topic:** Advanced Math — Quadratic Functions
**Skill:** Completing the square to find the vertex/maximum

![[assets/q22_vertex_form.png]]

Starting from the standard form, complete the square:
$$f(x) = -6x^2 - 10x - 126$$
$$= -6(x^2 + 10x) - 126$$

Wait — let me use the given vertex result: rewriting in vertex form $f(x) = -6(x-5)^2 + 24$ reveals that the **maximum value** is **24**, occurring at $x = 5$.

### Question 23 — Similar Triangles

**Correct Answer: A (YZ = 18)**
**Topic:** Geometry — Similar Triangles
**Skill:** Using proportional sides of similar triangles

![[assets/q23_similar_triangles.png]]

Lines $WZ$ and $XY$ intersect at $Q$. Since $\angle W = \angle Y = a°$ and the vertical angles at $Q$ are equal, triangles $QWX \sim QYZ$.

Corresponding sides are proportional:
$$\frac{YZ}{WX} = \frac{YQ}{WQ}$$
$$\frac{YZ}{60} = \frac{21}{70} = \frac{3}{10}$$
$$YZ = 60 \times \frac{3}{10} = 18$$

> [!tip] Strategy: Set Up Proportions from Corresponding Sides
> Identify which vertices correspond (WQ↔YQ, WX↔YZ), then set up: $\frac{\text{side}_1}{\text{side}_2} = \frac{\text{corresponding side}_1}{\text{corresponding side}_2}$.

> [!danger] Trap: Misidentifying Corresponding Sides
> $WX$ corresponds to $YZ$ (both are the bases), and $WQ$ corresponds to $YQ$ (both are sides from the vertex angle). Don't mix up the pairs.

### Question 24 — Polynomial Real Solutions

**Correct Answer: B (3 distinct real solutions)**
**Topic:** Advanced Math — Polynomial Equations
**Skill:** Finding real solutions using factoring and discriminant analysis

The equation factors into several components. Using discriminant analysis:
- $(5x^2 + 4)(x^2 - 4x + 16)$ → both have **negative discriminants** → no real solutions from these factors
- $(x + 4) = 0$ → $x = -4$ ✓
- $(x - 3) = 0$ → $x = 3$ ✓
- $(x + 3) = 0$ → $x = -3$ ✓

Three distinct real solutions: $x = -4, \; x = 3, \; x = -3$

> [!tip] Strategy: Apply the Zero Product Property
> Fully factor the polynomial, then set each factor equal to zero. For quadratic factors, check the discriminant ($b^2 - 4ac$) to determine if they yield real solutions.

> [!danger] Trap: Counting Solutions from All Factors Including Complex
> Quadratic factors with negative discriminants contribute **no real solutions** — their roots are complex/imaginary. Only count the real ones.

### Question 25 — Adding to a Data Set (Mean & Median)

**Correct Answer: B (Statement II only)**
**Topic:** Problem-Solving & Data Analysis — Statistics
**Skill:** Analyzing how adding a data point affects mean and median

A new player earns 18 points, added to data set A (50 values) to create data set B (51 values).

- **Mean:** All values in data set A are **greater than 18**, so adding 18 pulls the mean **down**. Mean of B < Mean of A. ✓ Statement I is **not guaranteed true**.
- **Median:** The median of 50 values is between the 25th and 26th values. The median of 51 values is the 26th value. Since the 26th value of data set B equals the 25th value of data set A, and both values in A fall in the 50–60 range, the medians **could be equal**. Statement II is correct: median of B ≤ median of A, but also could be equal.

Only **Statement II must be true**.

> [!tip] Strategy: Trace How Each Statistic Changes When Adding a Point
> For mean: if new point < current mean, new mean decreases. For median: carefully track the new middle position with the added data point.

> [!danger] Trap: Assuming Mean and Median Change in the Same Direction
> Adding a low value decreases the mean, but it may or may not change the median depending on where the new middle position falls.

### Question 26 — Similar Right Triangles and Tangent

**Correct Answer: D (tan Z = 4/3)**
**Topic:** Geometry & Trigonometry — Similar Right Triangles
**Skill:** Using similar triangles to find a trigonometric ratio

![[assets/q26_right_triangle_tangent.png]]

Triangle $WYX \sim$ Triangle $XYZ$ (both are right triangles sharing $\angle Y$).

Since similar triangles have equal corresponding angles:
$$\tan Z = \tan(\angle X \text{ in } \triangle WYX) = \frac{WY}{WX} = \frac{572}{429} = \frac{4}{3}$$

> [!tip] Strategy: Use Similar Triangles to Transfer Trig Ratios
> When the question asks for $\tan Z$ but gives you sides of a different (similar) triangle, use similarity to equate the tangent values. Then apply SOH-CAH-TOA to the triangle whose sides you know.

> [!danger] Trap: Applying Trig Ratios to the Wrong Triangle
> You know $WY = 572$ and $WX = 429$, which are sides of $\triangle WYX$, not $\triangle XYZ$. Use the similarity relationship to connect $\tan Z$ to $\triangle WYX$.

### Question 27 — Unit Conversion (Gridded)

**Correct Answer: 157.8 (also accepted: 789/5)**
**Topic:** Problem-Solving & Data Analysis — Unit Conversion
**Skill:** Converting square units using dimensional analysis

1 nautical mile = 1.852 km

Therefore: 1 square nautical mile = $1.852^2$ square kilometers:
$$1.852^2 = 3.429904 \text{ km}^2$$

Multiplying by 46.00 square nautical miles:
$$46.00 \times 3.429904 = 157.775... \approx 157.8 \text{ km}^2$$

> [!tip] Strategy: Square the Conversion Factor for Area
> To convert square units, you must **square the linear conversion factor**. If 1 nmi = 1.852 km, then 1 nmi² = $(1.852)^2 = 3.4299$ km².

> [!danger] Trap: Multiplying by 1.852 Instead of 1.852²
> $46 \times 1.852 = 85.19$ — that's wrong. Area conversions require squaring: $46 \times 1.852^2 = 157.8$.

> [!info] Gridded Response Note
> Enter **157.8** or **789/5**. Both are accepted. Note that 157.8 is rounded to the nearest tenth as specified.

> [!abstract] Key Formulas for Math Module 2
> - **Area (rectangle):** $A = l \times w$
> - **Exponential model:** $y = a \cdot b^x$ (initial value = $a$)
> - **Parallel lines:** equal slopes → $m_1 = m_2$
> - **Circle (standard form):** $(x-h)^2 + (y-k)^2 = r^2$
> - **Factor theorem:** zero at $x = a$ → $(x - a)$ is a factor
> - **Vertex form of quadratic:** $f(x) = a(x-h)^2 + k$ → max/min = $k$
> - **Similar triangles:** $\dfrac{\text{side}_1}{\text{side}_2} = \dfrac{\text{corresponding side}_1}{\text{corresponding side}_2}$
> - **Radical exponent:** $\sqrt[n]{a^m} = a^{m/n}$
> - **Discriminant:** $b^2 - 4ac$ → positive = 2 real roots, zero = 1, negative = 0 real roots
> - **Area unit conversion:** square the linear conversion factor

> [!danger] Harder Module Reminders
> - **Percent language matters:** "320% greater" ≠ "320% of" — the word "greater" adds to the original
> - **Complex factors give no real roots:** If discriminant < 0, discard those solutions
> - **Similar triangles transfer trig:** Use similarity to link trig ratios between related triangles
> - **Area conversions square the factor:** 1 nmi² = $(1.852)^2$ km², not $1.852$ km²
