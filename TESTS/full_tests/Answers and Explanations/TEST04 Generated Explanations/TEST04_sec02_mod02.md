### Q16 — ✅ **D) 139**

Given:
- $x = 6k + 13$
- $y = 8k - 29$
- lines \(m\) and \(n\) are parallel

At the top intersection, **\(x\)** and **\(y\)** are **vertical angles**, so they are equal:
$$6k + 13 = 8k - 29$$
$$42 = 2k$$
$$k = 21$$

Now find \(x\):
$$x = 6(21) + 13 = 126 + 13 = 139$$
But the question asks for $z$, not $x$.

Angle $z$ is **same-side interior** with $y$, so they are supplementary:

$z + y = 180$
Since $(y = x = 139)$:
$z = 180 - 139 = 41$

So the correct answer is:

$$41$$


> [!tip] Skill: Parallel Lines — Angle Relationships
> When a transversal cuts parallel lines: corresponding angles are equal, alternate interior angles are equal, and co-interior angles sum to 180°. Vertical angles at the same intersection are also equal. Identify the correct angle pair before setting up the equation.
> ``
> Use the angle relationships in the right order: first identify **vertical angles** to solve for the variable, then use **same-side interior angles** (or a supplementary pair) to get the requested angle.

### Q17 — ✅ **B) $\dfrac{1}{7}$**

### Q18 — ✅ **D) $-\dfrac{3}{2}$**

The function $f(x) = (x-10)(x+13)$ has roots at $x = 10$ and $x = -13$. The minimum (vertex) occurs at the midpoint of the roots:

$$x_{\text{vertex}} = \frac{10 + (-13)}{2} = \frac{-3}{2} = -\frac{3}{2}$$

> [!tip] Skill: Quadratic Functions — Vertex (Axis of Symmetry)
> For $f(x) = (x - r_1)(x - r_2)$, the vertex x-coordinate is the midpoint of the two roots: $x = \dfrac{r_1 + r_2}{2}$. This equals $-\dfrac{b}{2a}$ in standard form and is the axis of symmetry.

### Q19 — ✅ **A) The metal ball's minimum height was 3 inches above the ground**

The function is in vertex form $f(x) = \frac{1}{9}(x-7)^2 + 3$. The vertex is $(7, 3)$:
- $x = 7$: the time (in seconds) when the minimum occurs
- $y = 3$: the **minimum height** (3 inches above the ground)

## Q20 
**Correct answer:** $\dfrac{15}{17}$

**Given:**
- In triangle $JKL$, angle $J$ is a right angle
- $\cos(K) = \dfrac{24}{51}$

That means, relative to angle $K$:

$$\cos(K) = \frac{\text{adjacent}}{\text{hypotenuse}} = \frac{24}{51}$$

So one leg is $24$ and the hypotenuse is $51$.

Find the other leg with the Pythagorean theorem:

$$\sqrt{51^2 - 24^2} = \sqrt{2601 - 576} = \sqrt{2025} = 45$$

Now evaluate $\cos(L)$. Since $L$ is the other acute angle, the leg adjacent to $L$ is $45$:

$$\cos(L) = \frac{45}{51} = \frac{15}{17}$$

So the correct answer is:

$$\boxed{\frac{15}{17}}$$


> [!tip] Skill: Trigonometry — Complementary Angles in Right Triangles
> In a right triangle, the two acute angles are complementary ($K + L = 90°$). Therefore $\cos(L) = \sin(K)$. Use $\sin(K) = \frac{\text{opposite}}{\text{hypotenuse}}$ or apply the Pythagorean theorem to find the missing leg.

> [!note] Question Type: Right-triangle trigonometry
> When two acute angles are complementary, one angle’s cosine equals the other angle’s sine. If the question switches from \(K\) to \(L\), make sure you also switch which side is adjacent.

### Q21 — ✅ **51**

For $-x^2 + bx - 676 = 0$ to have **no real solutions**, the discriminant must be negative:

$$\Delta = b^2 - 4(-1)(-676) < 0$$
$$b^2 - 2{,}704 < 0$$
$$b^2 < 2{,}704$$
$$b < \sqrt{2{,}704} = 52$$

Since $b$ is a positive integer, the greatest possible value is $b = \mathbf{51}$.

> [!tip] Skill: Discriminant — No Real Solutions
> For $ax^2 + bx + c = 0$, no real solutions ↔ $b^2 - 4ac < 0$. Set up the strict inequality and solve for the boundary value. Since $b$ must be an integer, take the largest integer *strictly less than* the boundary.

### Q22 — ✅ **A) Zero**

From the graph, the two lines shown appear to intersect at a single point. The new line $x + 4y = -16$ can be rewritten as $y = -\frac{1}{4}x - 4$. 

Reading the graphed lines: one line has a negative slope passing through approximately $(0, 6)$ and $(8, 2)$ → slope $= \frac{2-6}{8-0} = -\frac{1}{2}$, equation roughly $y = -\frac{1}{2}x + 6$.

The new line $x + 4y = -16 \to y = -\frac{1}{4}x - 4$ has y-intercept $-4$ and slope $-\frac{1}{4}$. This line does **not** pass through the intersection point of the existing two lines. Since each pair of lines has a different intersection point, no single point satisfies all three equations simultaneously → **zero solutions**.

Check by testing whether the intersection of the original two lines satisfies the new equation.

### Q23 — ✅ **C) 36**

The function $f(x) = 5{,}470(0.64)^{x/12}$ represents monthly decay. For **yearly** decay, set $x = 12$:

$$f(12) = 5{,}470(0.64)^{12/12} = 5{,}470 \times 0.64$$

So after 1 year, the value is $64\%$ of the previous year's value — meaning it decreased by $100\% - 64\% = 36\%$.

Therefore $p = \mathbf{36}$.

> [!tip] Skill: Interpreting Exponential Decay Functions
> In $f(x) = A(r)^{x/k}$, the factor $r$ applies every $k$ units. Here $r = 0.64$ applies every 12 months (1 year). Since the value is multiplied by 0.64 each year, it *decreases* by $1 - 0.64 = 0.36 = 36\%$ per year.

Adding a constant (56) to every value in a data set:
- **Shifts** all values by the same amount → **median increases** by 56

> [!tip] Skill: Statistics — Effect of Adding a Constant

### Q25 — ✅ **D) $x^2 + (y+1)^2 = 49$**

Circle A: $x^2 + (y-1)^2 = 49$ has center $(0, 1)$.

Shifting **down 2 units** moves the center from $(0, 1)$ to $(0, -1)$.

New equation: $x^2 + (y - (-1))^2 = 49 \implies x^2 + (y+1)^2 = 49$

> [!tip] Skill: Transformations of Circles
> Standard form: $(x-h)^2 + (y-k)^2 = r^2$ has center $(h, k)$. Shifting **down** $n$ units subtracts $n$ from the $k$ value: new center $= (h,\ k - n)$. Be careful with the sign inside the parentheses.

### Q26 — ✅ **B) 8**

Let side length $= s$. For one rectangular prism (square base $s \times s$, height 90):

$$K = 2s^2 + 4(90s) = 2s^2 + 360s$$

When glued along a base, two square faces are hidden (one from each prism):

$$\text{New surface area} = 2K - 2s^2 = \frac{92}{47}K$$

Substitute $K = 2s^2 + 360s$ and expand:

$$47(2s^2 + 720s) = 92(2s^2 + 360s)$$
$$94s^2 + 33{,}840s = 184s^2 + 33{,}120s$$
$$720s = 90s^2$$
$$s = 8$$

> [!tip] Skill: Surface Area — 3D Geometry + Algebraic Setup
> Write the surface area formula for a rectangular prism with a square base ($K = 2s^2 + 4 \cdot \text{lateral face}$). When two prisms are glued base-to-base, subtract $2s^2$ (one hidden base from each). Set up and solve the resulting equation for $s$.

### Q27 — ✅ **600**

"$210$ is $p\%$ greater than $30$" means:

$$210 = 30 + \frac{p}{100} \cdot 30$$
$$180 = \frac{30p}{100}$$
$$p = \frac{180 \times 100}{30} = 600$$

> [!tip] Skill: Percent Increase
> "A is $p\%$ greater than B" → $A = B\left(1 + \dfrac{p}{100}\right)$, equivalently $A - B = \dfrac{p}{100} \cdot B$. Solve for $p$ by isolating it after substituting known values.
