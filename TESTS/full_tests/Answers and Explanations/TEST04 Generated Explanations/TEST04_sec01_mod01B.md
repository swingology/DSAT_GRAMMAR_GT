### Q14 — ✅ **80**

### Q22 — ✅ **B) 12**

### Q23 — ✅ **D) $\dfrac{45}{k}$**

From $(hx + k)(x + j) = 4x^2 + bx - 45$:
- Leading coefficient: $h = 4$
- Constant term: $kj = -45$

Since $k$ and $j$ are both integers and $kj = -45$, $k$ must be an integer divisor of 45. Therefore $\frac{45}{k}$ is always an integer. Meanwhile, $\frac{45}{h} = \frac{45}{4}$ is never an integer, and $\frac{b}{h} = \frac{4j+k}{4}$ is only an integer if $4 \mid k$, which isn't guaranteed.

> [!tip] Skill: Polynomial Factoring + Integer Divisibility
> Expand $(hx+k)(x+j)$ and match coefficients. Focus on the **constant term**: if $kj = -45$ and both are integers, then $k \mid 45$, guaranteeing $45/k$ is an integer.

### Q24 — ✅ **C) 6**

For one intersection point, the system has exactly one solution → discriminant $= 0$.

Set equal: $2x^2 - 21x + 64 = 3x + a \implies 2x^2 - 24x + (64 - a) = 0$

Discriminant $= 0$:

$$(-24)^2 - 4(2)(64 - a) = 0 \implies 576 - 8(64 - a) = 0 \implies 576 - 512 + 8a = 0 \implies a = -8$$

Substitute $a = -8$: $2x^2 - 24x + 72 = 0 \implies x^2 - 12x + 36 = 0 \implies (x-6)^2 = 0 \implies x = 6$

> [!tip] Skill: Systems of Equations + Discriminant
> "Intersects at exactly one point" = discriminant equals zero. Set the equations equal, rearrange to standard quadratic form, apply $b^2 - 4ac = 0$, solve for the constant, then solve for $x$.

### Q25 — ✅ **C) $58 + 58\sqrt{2}$**

In an isosceles right triangle: hypotenuse $= s\sqrt{2}$, so:

$$s\sqrt{2} = 58 \implies s = \frac{58}{\sqrt{2}} = 29\sqrt{2}$$

Perimeter $= 2s + \text{hypotenuse} = 2(29\sqrt{2}) + 58 = 58\sqrt{2} + 58 = 58 + 58\sqrt{2}$

> [!tip] Skill: 45-45-90 Special Right Triangle
> In a 45-45-90 triangle, legs $= s$ and hypotenuse $= s\sqrt{2}$. Given the hypotenuse, divide by $\sqrt{2}$ to find each leg, then compute the full perimeter.

### Q27 — ✅ **5**

$f(x) = -a^x + b$. The graph of $y = f(x) - 15$ has y-intercept $\left(0, -\frac{99}{7}\right)$.

At $x = 0$: 

$$f(0) - 15 = -a^0 + b - 15 = -1 + b - 15 = b - 16 = -\frac{99}{7}$$

$$b = -\frac{99}{7} + 16 = \frac{-99 + 112}{7} = \frac{13}{7}$$

Use the product condition $ab = \frac{65}{7}$:

$$a \cdot \frac{13}{7} = \frac{65}{7} \implies a = \frac{65}{13} = 5$$

> [!tip] Skill: Exponential Functions — Transformations + Systems
> A vertical shift $y = f(x) - 15$ subtracts 15 from all outputs. Plug in the y-intercept ($x=0$) to find $b$, remembering $a^0 = 1$ for any base. Then use the product condition to find $a$.
