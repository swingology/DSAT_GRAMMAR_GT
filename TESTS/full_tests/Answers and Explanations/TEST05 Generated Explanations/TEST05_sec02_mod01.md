## Question 19
**Answer: A) $-\sqrt{3}$**

> [!tip] Topic & Method
> **Topic:** Trigonometry — Evaluating Trig Functions / Periodicity
> **Method:** Reduce the angle using the period of tangent ($\pi$), then evaluate.

$$\tan\!\left(\frac{92\pi}{3}\right)$$

Since $\tan$ has period $\pi$, divide the angle by $\pi$:

$$\frac{92\pi/3}{\pi} = \frac{92}{3} = 30\tfrac{2}{3}$$

Subtract 30 full periods:

$$\frac{92\pi}{3} - 30\pi = \frac{92\pi - 90\pi}{3} = \frac{2\pi}{3}$$

$$\tan\!\left(\frac{2\pi}{3}\right) = \tan\!\left(\pi - \frac{\pi}{3}\right) = -\tan\!\left(\frac{\pi}{3}\right) = \boxed{-\sqrt{3}}$$

## Question 22
**Answer: B) −14**

> [!tip] Topic & Method
> **Topic:** Circles — Standard Form Equation
> **Method:** Use the equation of a circle to find the valid range of $x$-values (the $a$ values) for points on the circle.

The circle $(x + 4)^2 + (y - 19)^2 = 121$ has center $(-4, 19)$ and radius $r = \sqrt{121} = 11$.

For any point $(a, b)$ on the circle:

$$(a + 4)^2 \leq 121 \implies |a + 4| \leq 11 \implies -15 \leq a \leq 7$$

Check the choices:
- **A) $-16$:** $(-16+4)^2 = (-12)^2 = 144 > 121$ ✗
- **B) $-14$:** $(-14+4)^2 = (-10)^2 = 100 \leq 121$ ✓ (valid, e.g., $b = 19 \pm \sqrt{21}$)
- **C) $11$:** $(11+4)^2 = 225 > 121$ ✗
- **D) $19$:** way out of range ✗

Answer: $a = \boxed{-14}$

## Question 24
**Answer: D) Neither I nor II**

> [!tip] Topic & Method
> **Topic:** Exponential Functions — Behavior and Asymptotes
> **Method:** Analyze whether each function achieves a minimum value. Exponential functions with a negative coefficient are strictly decreasing with no minimum.

**Function I:** $f(x) = -6(3)^x - 3$

Since $(3)^x > 0$ for all $x$, we have $-6(3)^x < 0$, so $f(x) < -3$ for all $x$.
- As $x \to -\infty$: $f(x) \to -3$ (asymptote, never reached)
- As $x \to +\infty$: $f(x) \to -\infty$

$f$ has **no minimum** (unbounded below). $-3$ is a horizontal asymptote, not a minimum.

**Function II:** $g(x) = -3(6)^x$

- As $x \to -\infty$: $g(x) \to 0$ (asymptote)
- As $x \to +\infty$: $g(x) \to -\infty$

$g$ also has **no minimum**. The value $-3$ is never achieved as a minimum.

**Neither function has a minimum value at $-3$.**

## Question 26
**Answer: A) −6**

> [!tip] Topic & Method
> **Topic:** Quadratic Functions — Factored Form and Coefficients
> **Method:** Use the two roots to write $f$ in factored form, then expand to find $a + b$.

Since $f(7) = 0$ and $f(-3) = 0$, the roots are $x = 7$ and $x = -3$:

$$f(x) = a(x - 7)(x + 3)$$

Expand:

$$f(x) = a(x^2 - 4x - 21) = ax^2 - 4ax - 21a$$

So $b = -4a$ and:

$$a + b = a + (-4a) = -3a$$

Since $a$ is an integer greater than 1, the smallest value is $a = 2$:

$$a + b = -3(2) = \boxed{-6}$$

For $a = 3$: $a + b = -9$, for $a = 4$: $a + b = -12$, etc. Only $-6$ appears among the choices.

## Question 27
**Answer: 25**

> [!tip] Topic & Method
> **Topic:** Polynomial Functions — Zeros and Solving Equations
> **Method:** Set $7 - w$ equal to each zero of $g(x)$ and solve for $w$, then sum all solutions.

$g(x) = x(x-2)(x+6)^2 = 0$ when:

$$x = 0, \quad x = 2, \quad x = -6$$

Since $g(7 - w) = 0$, set $7 - w$ equal to each zero:

| Zero | Equation | $w$ |
|------|----------|-----|
| $0$ | $7 - w = 0$ | $w = 7$ |
| $2$ | $7 - w = 2$ | $w = 5$ |
| $-6$ | $7 - w = -6$ | $w = 13$ |

$$\text{Sum} = 7 + 5 + 13 = \boxed{25}$$
